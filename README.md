Venom
=====

Simple vendoring management tool for PHP 5.2

Currently autoloader does not work.

How to use
----------

Currently Venom can install only from GitHub repo.

Save below as `Venomfile`

```php
<?php
github('https://github.com/codeguy/Slim');
```

And execute below.

```
$ php venom.php
```
